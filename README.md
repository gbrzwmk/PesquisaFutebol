# PesquisaFutebol
Projeto web Python com fluxo de 3 etapas. O usuário faz um cadastro validado, depois um quiz de futebol e, por fim, uma votação. O sistema usa a sessão do Flask para controlar o progresso, impedindo que o usuário pule etapas e que vote mais de uma vez. O código é modular, separando o servidor da lógica de validação e do quiz.
